# Summer Internship Verilog Codes

A collection of Verilog HDL projects developed during my Summer Research Internship and personal learning in Digital Design and Computer Architecture.

The repository contains implementations of combinational circuits, sequential circuits, arithmetic units, finite state machines, multipliers, floating-point units, and other digital hardware modules. Each project is organized in its own directory with source code, testbenches, and supporting files where applicable.

## Repository Contents

Most project folders contain:

- Verilog source files
- Testbench files
- Vivado project
- Constraint files (where applicable)
- Simulation outputs

Generated Vivado files such as `.runs`, `.sim`, `.cache`, and other temporary build artifacts are intentionally excluded from this repository.

---

## Running a Project

1. Clone the repository.

```
git clone https://github.com/Saksham0126/Summer-Internship-Verilog-Codes.git
```

2. Open the desired project in Vivado.

3. Run Behavioral Simulation.

4. Modify or extend the design as required.
