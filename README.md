## 📖 About

This repository contains the Verilog HDL source code, simulation files, reports, and project documentation developed during my internship. The projects cover various aspects of digital system design, combinational and sequential logic, arithmetic circuits, and FPGA implementation.

---

## 📂 Projects

* 🔹 Floating Point Adder (IEEE-754 Single Precision)
* 🔹 Booth Multiplier
* 🔹 Linear Carry Select Adder (LCSA)
